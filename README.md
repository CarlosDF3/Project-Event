Este proyecto fue desarrollado inicialmente como una primera experiencia de trabajo en equipo, con el objetivo de introducirnos en el desarrollo web full-stack utilizando PHP, MySQL, JavaScript y tecnologías frontend básicas.

Tras la fase inicial en grupo, el proyecto fue retomado individualmente para realizar mejoras en la interfaz, refinar algunos comportamientos del frontend y ajustar determinadas partes de la lógica.

Dado su carácter formativo y su evolución progresiva, el código presenta una estructura mejorable y ciertas decisiones de implementación propias de una etapa inicial de aprendizaje, especialmente en la organización general y la consistencia de algunas funcionalidades.

Aun así, el proyecto sirvió como una base sólida para comprender el flujo completo de una aplicación web y la interacción entre cliente, servidor y base de datos.
